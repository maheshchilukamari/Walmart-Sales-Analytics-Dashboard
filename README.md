# Walmart Sales Analytics Dashboard

An interactive data visualization dashboard developed to analyze Walmart weekly sales, economic indicators, seasonal patterns, and holiday performance.

The project uses Python, Pandas, Plotly, and Dash to transform retail sales data into interactive charts and business insights.

## Project Overview

The dashboard enables users to explore:

- Weekly sales performance
- Holiday and non-holiday sales trends
- Store-level sales comparisons
- Temperature, fuel price, CPI, and unemployment relationships
- High-performing and low-performing sales periods
- Correlations between economic indicators and weekly sales

## Technologies Used

- Python
- Pandas
- Plotly
- Dash
- Dash Bootstrap Components
- Jupyter Notebook
- HTML and CSS
- Git and GitHub
